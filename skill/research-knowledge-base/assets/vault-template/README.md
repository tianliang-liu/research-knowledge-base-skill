# 我的研究知识库

这个文件夹是一个独立研究知识库，用来沉淀经过 Zotero 收集、Codex 辅助精读、Obsidian 整合后的长期研究知识。

请先根据自己的方向修改：

- 知识库名称
- 研究方向
- 项目页
- 主题页
- 方法概念页
- 标签体系
- Zotero collection

## 文件夹结构

```text
我的研究知识库
├── 01_Literature
├── 02_Projects
├── 03_Topics
├── 04_Methods_Concepts
├── 05_Synthesis
└── 90_Templates
```

## 核心原则

1. Zotero 负责文献实体和 PDF。
2. Obsidian 负责精读后的研究知识。
3. Codex 负责辅助读取、拆解、总结和写入。
4. 每篇重要文献只在 `01_Literature` 中形成一篇正式笔记。
5. 新项目通过链接和证据地图复用已有文献，不重复全文整理。
6. 单篇文献笔记完成后，要沉淀到项目页、主题页、方法页或综合页。

## 模板设置

Obsidian 模板文件夹设置为：

```text
我的研究知识库/90_Templates
```

## 文献状态

```text
unread       已收集，还没读
reading      正在读
summarized   已完成单篇文献笔记
synthesized  已沉淀到上层页面
archived     暂时没有价值
```
